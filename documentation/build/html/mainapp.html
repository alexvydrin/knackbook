Основное приложение¶

Приложение, которое отвечает за:

  • главную страницу

  • список статей

  • детали статьи

  • обратную связь

  • поиск и др

Контроллеры mainapp¶

Контроллеры (Views)

class mainapp.views.ArticlesForSearch(**kwargs)¶

Просмотр списка статей для введенного в строку поиска текста, с учетом расширенного поиска, при котором добавляются отборы по дате изменения статьи и рейтингу статьи (при его активации)

get_context_data(**kwargs)¶

Get the context for this view.

get_queryset()¶

Return the list of items for this view.

The return value must be an iterable and may be an instance of QuerySet in which case QuerySet specific behavior will be enabled.
